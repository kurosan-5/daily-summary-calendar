# 1日行動サマリーカレンダー

毎日の行動を記録して、AIが自動で要約・スコア化するアプリケーションです。

## 機能

- **今日の行動記録**: テキストエリアに自由記述で行動を入力
- **AI自動分析**: Google Gemini AIを使用して要約・10段階スコア・タグ・場所を自動生成
- **カレンダー表示**: 月別カレンダーでスコアを色で可視化
- **食事チェック**: 朝昼夜の食事状況を記録
- **外出ヒートマップ**: 月間の外出度を視覚化
- **日別詳細**: AI要約、フィードバック、タグ、場所などの詳細表示

```js
      },
      // other options...
    },
  },

```
